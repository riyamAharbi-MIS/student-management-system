#  Student Management System

##  Project Description
This project is a simple Student Management System built using SQL.
It manages students, courses, and enrollments.

##  Technologies Used
- SQL
- Database Design

## Features
- Store student information
- Manage courses
- Track enrollments and grades
- Perform JOIN queries

##  What I Learned
- Database design
- Creating tables with relationships
- Writing SQL queries and joins
## Results
The project successfully displays:
- Student records
- Course data
- Enrollment relationships using JOIN queries
